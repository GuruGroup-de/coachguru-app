# CoachGuru Documentation

This folder contains the GitHub Pages website for CoachGuru.

## GitHub Pages Configuration

To enable GitHub Pages using this `/docs` folder:

1. Go to your repository settings on GitHub
2. Navigate to **Pages** section
3. Under **Source**, select **Deploy from a branch**
4. Select **main** branch
5. Select **/docs** folder
6. Click **Save**

Your website will be available at:
`https://GuruGroup-de.github.io/coachguru-app/`

## Files

- `index.html` - Main landing page for CoachGuru

## Customization

You can update the `index.html` file to:
- Add actual APK download link (currently points to GitHub Releases)
- Update contact information
- Add more features or sections
- Customize colors and styling
