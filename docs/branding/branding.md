# CoachGuru Brand Guidelines

This document outlines the brand identity, color palette, typography, and logo usage guidelines for CoachGuru.

---

## üé® Color Palette

### Primary Colors

CoachGuru uses a blue-white gradient theme that represents professionalism, trust, and clarity.

#### Main Blue
- **Hex**: `#4A90E2`
- **RGB**: `74, 144, 226`
- **Usage**: Primary buttons, links, highlights, logo

#### Dark Blue
- **Hex**: `#0288D1`
- **RGB**: `2, 136, 209`
- **Usage**: Hover states, gradients, emphasis

#### Light Blue
- **Hex**: `#E3F2FD`
- **RGB**: `227, 242, 253`
- **Usage**: Backgrounds, subtle highlights, cards

### Accent Colors

#### Accent Green
- **Hex**: `#4CAF50`
- **RGB**: `76, 175, 80`
- **Usage**: Success states, positive actions, confirmations

#### Error Red
- **Hex**: `#E53935`
- **RGB**: `229, 57, 53`
- **Usage**: Error messages, delete actions, warnings

#### Warning Orange
- **Hex**: `#FF9800`
- **RGB**: `255, 152, 0`
- **Usage**: Warnings, alerts, important notices

### Neutral Colors

#### Text Dark
- **Hex**: `#1A1A1A`
- **RGB**: `26, 26, 26`
- **Usage**: Primary text, headings, important content

#### Text Light
- **Hex**: `#666666`
- **RGB**: `102, 102, 102`
- **Usage**: Secondary text, descriptions, subtitles

#### Soft Grey
- **Hex**: `#F5F5F5`
- **RGB**: `245, 245, 245`
- **Usage**: Backgrounds, dividers, subtle elements

#### White
- **Hex**: `#FFFFFF`
- **RGB**: `255, 255, 255`
- **Usage**: Cards, backgrounds, text on colored backgrounds

### Gradient Combinations

#### Primary Gradient
```
Linear Gradient (135deg):
- Start: #4A90E2 (Main Blue)
- End: #0288D1 (Dark Blue)
```

#### Background Gradient
```
Linear Gradient (135deg):
- Start: #E3F2FD (Light Blue)
```

## üìù Typography

### Font Family

CoachGuru uses system fonts for optimal performance and native feel:

**Primary Font Stack:**
```
-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif
```

### Type Scale

#### Headings

- **H1 (Hero Title)**: `3rem` / `48px` - Bold (700)
  - Usage: Main page titles, hero sections
  - Color: `#1A1A1A` (Text Dark)

- **H2 (Section Title)**: `2.5rem` / `40px` - Bold (700)
  - Usage: Section headings, major content divisions
  - Color: `#1A1A1A` (Text Dark) or `#4A90E2` (Main Blue)

- **H3 (Card Title)**: `1.5rem` / `24px` - Semi-bold (600)
  - Usage: Card titles, feature names
  - Color: `#1A1A1A` (Text Dark)

#### Body Text

- **Large**: `1.3rem` / `20.8px` - Regular (400)
  - Usage: Hero subtitles, important descriptions

- **Regular**: `1rem` / `16px` - Regular (400)
  - Usage: Body text, descriptions, content
  - Color: `#666666` (Text Light)

- **Small**: `0.95rem` / `15.2px` - Regular (400)
  - Usage: Captions, metadata, secondary information

#### Buttons

- **Button Text**: `1.1rem` / `17.6px` - Semi-bold (600)
  - Usage: All button labels
  - Color: `#FFFFFF` (White) on colored backgrounds

### Line Height

- **Headings**: `1.2` - `1.3`
- **Body Text**: `1.6` - `1.8`
- **Buttons**: `1.0`

### Letter Spacing

- **Logo**: `-2px` to `-0.5px` (tight)
- **Headings**: `-1px` to `0px` (slightly tight)
- **Body**: `0px` (normal)

## üéØ Logo Usage

### Logo Files

- **Horizontal Logo**: `coachguru-logo.png` (800x200px)
  - Use for: Headers, documentation, wide spaces
  
- **Square Logo**: `coachguru-logo-square.png` (400x400px)
  - Use for: App icons, social media profiles, square spaces

### Logo Placement

#### Minimum Size
- Horizontal logo: Minimum width `200px`
- Square logo: Minimum size `100x100px`

#### Clear Space
- Maintain clear space equal to **50% of the logo height** on all sides
- Never place text or graphics within this clear space

#### Background Usage

**Light Backgrounds** (White, Light Blue, Soft Grey):
- Use full-color logo with Main Blue (`#4A90E2`)

**Dark Backgrounds** (Text Dark, Dark Blue):
- Use white or light-colored logo variant
- Ensure sufficient contrast (minimum 4.5:1)

**Colored Backgrounds**:
- Use white logo variant
- Avoid placing on backgrounds too similar to logo color

### Logo Don'ts

‚ùå **Do NOT:**
- Stretch or distort the logo
- Rotate the logo
- Add effects (shadows, outlines, gradients) to the logo
- Place logo on busy backgrounds
- Use logo colors other than brand colors
- Modify logo proportions
- Use logo at sizes smaller than minimum

‚úÖ **Do:**
- Maintain aspect ratio
- Use high-resolution versions
- Ensure proper contrast
- Follow clear space guidelines
- Use appropriate logo variant for context

## üé® Design Principles

2. **Professional**: Business-appropriate aesthetics
3. **Accessible**: High contrast, readable typography

- **Element Gap**: `15px - 30px`
